# 🔌 Portable AC Power System (220V Output)

## 🚀 Overview

This project presents a **portable AC power system** capable of delivering **220V AC output** from a Li-ion battery pack, designed as a **mobile electricity solution** for real-world applications.

The system integrates **battery management, power conversion, and safety mechanisms** into a compact and standalone unit.

---

## ⚙️ System Architecture

- 🔋 Power Source: Li-ion Battery Pack  
- ⚡ Output: 220V AC  
- 🔌 Charging: Integrated 12.6V charging system (plug-and-charge)  
- 🧠 Control: Embedded monitoring and protection logic  

---

## 🔥 Key Features

✔ 220V AC output (portable electricity system)  
✔ Integrated internal charging system  
✔ Complete protection system (BMS + load protection)  
✔ Standalone energy unit (no external systems required)  
✔ Real-world usable hardware design  

---

## 🧠 Engineering Design

### ⚡ Power Conversion

- Converted battery DC power into **220V AC output**  
- Designed system for stable and safe AC delivery  
- Ensured compatibility with real-world electrical loads  

---

### 🔋 Charging System

- Integrated **12.6V charging system** directly into the device  
- Enabled **direct AC plug-in charging**  
- Implemented safe charging control to prevent overcharging  

---

### 🛡️ Protection Mechanisms

- Overcharge protection  
- Over-discharge protection  
- Overcurrent protection (load safety)  
- Safe switching using control logic and hardware protection  

---

### 📊 Monitoring System

- Developed internal monitoring to track battery voltage and system status  
- Ensured stable operation and voltage regulation  
- Integrated visual feedback for user interaction  

---
